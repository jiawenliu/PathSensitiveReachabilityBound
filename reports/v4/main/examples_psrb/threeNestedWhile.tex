  The bound on the maximum value of each ranking function, i.e., the \textbf{ranking function bound} is as follows,
  \\
  $\varinvar(i) = n$  \quad
  $\varinvar(j) = m * n$  \quad
  $\varinvar(w) = n$  \quad
  $\varinvar(w - n + N) = N$
